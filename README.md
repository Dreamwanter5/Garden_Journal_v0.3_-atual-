# Introdução

O presente repositório contém os arquivos do Garden Journal, um projeto desenvolvido por mim, Pedro Luis, um estudante do quarto ano de informática, findando a conclusão do ensino médio, que para isso, precisa de um TCC (Trabalho de Conclusão de Curso.)


# Sobre o projeto

Tendo esta premissa, o Garden Journal pressupõe-se como uma aplicação web que funciona de forma análoga a outros aplicativos como o Obsidian.md e o Notion feito pela NotionLabs.inc, deste modo, este projeto agiria como um organizador de diferentes anotações, capazes de estabelecerem ligações entre si, cada uma delas podendo portar *tags* que tornem os arquivos facilmente identificáveis e heterogêneos mediante todas as outras anotações.

Este projeto surgiu com a premissa de que a sociedade que se desenvolveu no século XXI se tornou muito dependente da tecnologia (Jornal da USP, 2024), passando a maioria de suas horas acordadas em frente às telas, visto isso, a quantidade de informações consumidas diariamente de forma torrencial se tornou muito exacerbada, atrasando assim a realização de atividades casuais e profissionais dos indivíduos. O Garden Journal advém deste cenário com o propósito de ser um organizador de ideias neste mundo caótico do século XXI.

# Conclusão

O projeto por si só foi um fim em si próprio, com a sua finalidade sendo a de servir como ferramenta de aprendizado para o estudante que o desenvolveu. Sendo um conglomerado das mais diversas lógicas que existe dentro da área da programação. Isto, contendo diferentes linguagens de programação como HTML, PHP, CSS, Javascript e SQL, isto sem contar outras importantes funcionalidades, em especial as implementações de plugins, entendimento de funções assíncronas, orientação a objetos e afins...
